## 🔹 Project 1: Perceptron Model

### Objective
To implement a perceptron model using Python.

### Results
- AND Gate: 100% accuracy
- OR Gate: 100% accuracy
- XOR Gate: Failed (non-linear problem)

### Conclusion
Perceptron works only for linearly separable data.

---

## 🔹 Project 2: Competitive Learning Network

### Objective
To implement a competitive learning network that can self-organize and cluster input data.

### Method
- Generated noisy 2D data
- Used winner-takes-all approach
- Updated only the winning neuron

### Results
- Neurons successfully formed clusters
- Each neuron specialized for a region

### Challenges
- Choosing number of neurons
- Sensitivity to initial weights

---

## 🔹 Project 3: Feedforward Neural Network

### Objective
To design a feedforward neural network for classification.

### Method
- Used one hidden layer
- Activation functions: ReLU and Sigmoid
- Trained using backpropagation

### Results
- Model successfully classified input data
- Achieved good accuracy

### Conclusion
FNN can solve complex non-linear problems better than perceptron.

---

## 🏁 Final Conclusion
All three models demonstrate different learning techniques:
- Perceptron → Simple linear classification  
- Competitive Learning → Unsupervised clustering  
- Feedforward NN → Complex pattern recognition
